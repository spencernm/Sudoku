# Sudoku Puzzle Generator

Spencer Chin

This program generates a randomly solved 9x9 Sudoku board.

Run the program in the terminal.

The Sudoku board is generated by created a list of "seed numbers" which will influence the turnout of the solution. The seed numbers go through checks to ensure the rules of Sudoku are met, and the program ends when a 9x9 Sudoku board is generated.
